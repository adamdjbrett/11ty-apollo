# apollo

Modern and minimalistic blog theme powered by [Eleventy](https://11ty.dev), ported from the original [Zola](https://getzola.org) theme.

- Original theme: [not-matthias/apollo](https://github.com/not-matthias/apollo)

- Demo: <https://not-matthias.github.io/apollo>

## Local Development

```bash
npm install
npm run start
```

## Production Build

```bash
npm run build
```
